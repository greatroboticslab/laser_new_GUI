# Overview
This project reads displacement from laser and piezo onto umd2 microcontorller, and displays data in a graphical format

# Install & Run

### Linux/ MacOS
First you must create the script (only have to do once):

- ```chmod +x run.sh```

To run program:

- ```./run```

### Windows
First you must create the script (only have to do once):

- ```powershell -ExecutionPolicy Bypass -File .\run.ps1```


To run program:

- ```.\run```
